# weather-js

Простое приложение для просмотра погоды в выбранном городе.

Сервис работает благодаря API от [Weatherbit.io](https://www.weatherbit.io/)

## Технологии
- HTML/CSS
- Bootstrap
- JS (ES6 Classes, Fetch API)
